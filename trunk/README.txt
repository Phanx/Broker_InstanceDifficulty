Broker Instance Difficulty
=============================

Shows the current instance difficulty on your Broker display.

There are no options. By default, no text is shown when you are not in
a dungeon, raid, or scenario; if you want to see some text, open the
Addon.lua file in Notepad and edit the DEFAULT_TEXT value at the top.

Localization
---------------

Compatible with all languages.

If the C/H/N/LFR/FLEX abbreviations are not widely understood in your
language, please post a comment with your preferred abbreviations.
